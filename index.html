// HOW TO CONNECT

Pick your device.
Play in minutes.

Choose your platform below and follow the steps to connect to Arcade-PS.

🪟
WINDOWS

Edit your hosts file — copy the entries below and paste them in.

  1. Press Win + R, type notepad, right-click Notepad → Run as administrator.
  2. Open C:\Windows\System32\drivers\etc\hosts.
  3. Copy the entries below and paste at the bottom of the file, then save.
  4. Open Growtopia and start playing!
✓ Copied! Paste into your hosts file.
🤖
ANDROID

Download the hosts file or copy the link to use with a hosts editor app.

  1. Download arcade-hosts.txt or copy the hosts link below.
  2. Use a hosts editor app (e.g. Hosts Editor) and import the file.
  3. Fully close Growtopia before connecting.
  4. Open Growtopia — it connects automatically.
⬇ Download hosts.txt
🍎
IOS

Use Surge 5 and the Arcade-PS profile URL.

  1. Download Surge 5 from the App Store.
  2. Open Surge 5 → tap Default.conf.
  3. Go to Import → Download Profile from URL, enter /ios.
  4. Tap OK → Done → Setup → allow VPN config.
  5. Open Growtopia and enter ARCADE-PS to connect.
App Store ↗
💻
MACOS

Edit the hosts file manually — takes under 2 minutes.

  1. In Finder → Go → Go to Folder → type /private/etc/hosts.
  2. Drag the hosts file to your Desktop.
  3. Open with TextEdit, paste the copied entries at the bottom and save.
  4. Drag the file back to /private/etc/ and replace the old one.
  5. Open Growtopia and connect!

// FAQ

Frequently asked questions.

How do I switch back to official Growtopia servers?

On Windows, run GTPS.bat again — it will uninstall the changes. On Android, disconnect via the GTPS.link app. On iOS, disable the Surge VPN profile. On macOS, remove the two lines you added from the hosts file.

Where can I get help?

Join our Discord server and post in the support channels. Staff and community members are active daily.

How do I recover my account?

Type -recover in the #bot-commands channel on Discord and follow the bot's prompts to secure your GrowID.

How do I deposit items or top up?

Use the /deposit command in-game, or visit our purchase hub for all deposit options.

Is Arcade-PS free to play?

Yes! Arcade-PS is completely free to join and play. Optional cosmetics and items are available in the shop for those who want to support the server.

Is this affiliated with the official Growtopia or Ubisoft?

No. Arcade-PS is a fan-made private server and is not affiliated with Ubisoft or the official Growtopia game in any way.
